Ana Iglesias Arias

Me encanta resolver problemas complejos y encontrar soluciones creativas. De esta manera disfruto desafiándome a mí misma con el cubo de Rubik, aplicando ese mismo enfoque a mis proyectos de desarrollo web. Me encanta transformar ideas en realidades digitales, creando sitios web responsive, modernos y funcionales. Uso herramientas como Figma para diseñar interfaces atractivas y Bootstrap para convertir esos diseños en experiencias web reales y accesibles. Al igual que con el cubo de Rubik, donde cada movimiento debe ser preciso para alcanzar el objetivo, en mis proyectos busco la perfección en cada línea de código, asegurando que todo funcione a la perfección en cualquier dispositivo. ¡Vamos a resolver juntos este rompecabezas digital!
